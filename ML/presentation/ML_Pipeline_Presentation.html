Conclusions

Key Findings

  • Best Model: Decision Tree with LOOCV achieves 90.3% accuracy
  • Best AUC: Random Forest with LOOCV achieves 0.945 AUC
  • Most Important Features: Language detection (bio_language, name_language, location_language)
  • TF-IDF Value: Marginal impact - language features alone are highly predictive
  • Why It Works: Strong correlation between writing language and locality (65.4% for Russian)

Recommendations

  • Use Decision Tree or XGBoost for best accuracy
  • Use Random Forest for best AUC (0.945)
  • Language features alone are sufficient - TF-IDF optional
  • Use LOOCV for thorough evaluation on small datasets
